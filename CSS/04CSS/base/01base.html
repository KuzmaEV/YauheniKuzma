Форматы файлов.

.DOC или .doc (от англ. document) - расширение имени файла, использумое для файлов, представляющих текст, с разметкой или без. Асширение .DOC часто использовалось для обозначения простых текстовых файлов без формирования, однако позже стало использоваться для двойных форматов с разметкой.
Документ MS Word

Portable Document Format ( PDF ) - межплатформенный формат электронных документов, разработаннй фирмой Adobe Systems с использованием ряда возможностей языка Postscript.
Документ PDF

ZIP - популярный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет расширение .zipи хранит в сжатом или несжатом виде один или несколько файлов, которые можно из него извлеч путем распаковки с помощью специальной программы.
